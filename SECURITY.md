# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, email **sjoerdvink99@gmail.com** with the subject line `[SECURITY] Tableverse`. Include:

- A description of the vulnerability and its potential impact
- Steps to reproduce or a proof-of-concept
- Any suggested fix if you have one

You will receive an acknowledgement within 48 hours and a resolution timeline within 5 business days. Once a fix is released, you will be credited in the changelog unless you prefer to remain anonymous.
